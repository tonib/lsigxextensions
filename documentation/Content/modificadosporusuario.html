Objetos modificados por un usuario desde una fecha

Permite buscar objetos modificados por un usuario desde una fecha, aunque estos objetos hayan sido modificados posteriormente por otro usuario.

La extensión está en Lsi.Extensions > Toolwindows > Objects modified by users

La extensión pide un usuario y una fecha. El usuario puede ser una parte del nombre real del usuario Windows. P.ej. si el nombre completo del usuario es "DOMINIO\usuariox", si se indica "usuariox", los objetos se encontrarán.

La extensión revisa el historial de modificaciones de cada objeto modificado desde la fecha indicada, inclusive. Si el historial de modificaciones del objeto desde la fecha indicada contiene una modifiación del usuario indicado, se reporta el objeto.
