Pegar como literal de texto

Permite pegar un texto copiado desde otro programa en el editor de código de Genexus como un literal string. Esto puede ser útil para pegar código HTML a incluir en el objeto. SÓLO está soportado copiar texto plano. No está soportado copiar, p.ej., texto con formato desde Microsoft Word.

Se tiene en cuenta si el texto pegado contiene comillas simples o dobles.

La extensión está en Lsi.Extensions > Edit > Paste as GeneXus string literal. Un texto copiado como el siguiente: 

        <p>
            <b>Fecha y hora prevista de inicio del transporte:</b> 31/01/2019 09:10
        </p>

Se pega como esto: 

'        
' + NewLine() + 
'            Fecha y hora prevista de inicio del transporte: 31/01/2019 09:10' + NewLine() + 
'        
'
